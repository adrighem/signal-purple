# Support

This is a volunteer, pre-alpha project.

- Use GitHub Discussions for usage and design questions when enabled.
- Use GitHub Issues for reproducible bugs and feature proposals.
- Use private vulnerability reporting for security problems.

Before filing a bug, include the signal-purple revision, distro, Pidgin and
libpurple versions, Rust version, build command, and sanitized debug output.
Never include message contents, phone numbers, service identifiers, QR
provisioning URIs, keys, or the SQLCipher passphrase.
