# Support

This is a volunteer, pre-alpha project.

- Read the [README](README.md) and
  [troubleshooting guide](docs/troubleshooting.md) first. GitHub Discussions is
  not currently enabled, so general usage support is not currently offered
  there.
- Use GitHub Issues for reproducible bugs and feature proposals.
- Use private vulnerability reporting for security problems.

Before filing a bug, include the signal-purple revision, installation scope,
distro, Pidgin and libpurple versions, Rust version when built from source,
build command or package identity, loaded plugin/backend paths, and sanitized
debug output. Never include message contents, phone numbers, service
identifiers, QR provisioning URIs, keys, or the SQLCipher passphrase.
